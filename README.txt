== Description ==



== Installation ==

1. Upload the 'restaurants' folder to the '/wp-content/plugins/'' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `<?php do_action('restaurants_hook'); ?>` in your templates
